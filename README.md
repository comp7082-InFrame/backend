# Welcome to InFrame backend
This repo provides restful APIs that the frontend will utilize.
## Tech
FastAPI, Python, and some face detection model
## Try it
1. clone or fork repo
2. cd backend
3. py -3.11 -m venv venv
4. venv\Scripts\activate
5. source venv/bin/activate
6. pip install -r requirements.txt
7. uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

1. scan scene
